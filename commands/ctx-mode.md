---
name: ctx-mode
description: Switch context loading mode between smart and full — usage: /ctx-mode smart|full
---

`~/.cowork-memory/settings.json` の `context_mode` を "$ARGUMENTS" に変更してください。

有効な値: `smart` または `full`

変更後に現在の設定を表示してください：
```
[Mode changed: context_mode = "<new_mode>"]
```

"$ARGUMENTS" が空または無効な場合は、現在の設定を表示するだけにしてください。
